Transformationen

Idee: die optische Gestalt von Elementen transformieren.

CSS-Eigenschaft: transform

Werte, Tranformationsmatrizen, die von CSS-Funktionen erstellt werden, um bestimmte Veränderungen am Element zu bewirken.

  • Skalieren: scale(Faktor), optional nur in eine Dimention mit scaleX(Faktor) oder scaleY(Faktor)
  • Verschieben: translate(Länge für X-Verschiebung, Länge für Y-Verschiebung). %-Angaben beziehen sich auf das Element selbst, nicht auf das Elter
  • Drehen: rotate(Winkel), rotate ohne Angaben meint um die Z-Achse (die Linie vom Bildschirm in unser Gesicht.) positive Winkel gehen in Richtung einer Achse gegen den Uhrzeiger-Sinn.
  • Schrägstellen (wie font-style:italic): skew(Winkel) Schräg oder Schiefstellen entlang der X-Achse.

die 3. Dimension

Für Alle Transformationen die in Z-Richtung gehen sollen, muss der Browser wissen, wie weit wir als Betrachter von der Webseite weg sind.

Wir nennen dem Browser den Abstand zwischen uns und dem Bildschirm über zwei Möglichkeiten.

  • perspective-Funktion innerhalb der transform eigenschaft. 
    transform: perspective(500px) translateZ(200px);
  • CSS-Eigenschaft: perspective an einem Elternelement, das gilt dann für alle Kinder, die was in Z-Richtung machen. 
    .3Dcontext 
{ 
  perspective: 500px 
}

.3Dcontext .entgegen {
  transform: translateZ(300px);
}

Beispiele

Ausgangspunkt von Transformationen

engl. die transform-origin, kann über CSS auch neu gesetzt werden. Es handelt sich um die Position der koordinate (0,0), den Schnittpunkt von X- und Y-Achse innerhalb des Elements.

Standard ist Mitte. transform-origin: center center;
